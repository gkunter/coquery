Coquery is a free corpus query tool for linguists, lexicographers, translators, and anybody who wishes to search and analyse a text corpus. The program is available for Windows, Linux, and Mac OS X computers.

News

November 30, 2021: Installer files for second preview of the next version 0.11.0 are available for Windows and Mac OS.

October 31, 2017: Installer files for a preview of the next version 0.11.0 are available for Windows and Mac OS.

May 23, 2017: Version 0.10.0 "ICAME38" is now available. This version is a major upgrade over the previous versions, reflecting nearly a year of development. See the Change Log for an overview.

September 21, 2016: Version 0.9.2a is now available for Windows. This version fixes an issue with the desktop shortcut.

May 1, 2016: Version 0.9.2 is now available. New features:

  • Corpora can now be built from .docx, .odt, HTML, or plain text files
  • Query results from speech corpora can now be saved as Praat TextGrids
  • Support for new corpora: Switchboard, Brown
